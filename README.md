# 📊 Netflix Content Analysis Dashboard (Power BI)

## 📌 Overview
This project presents an interactive Power BI dashboard analyzing Netflix content across genres, countries, ratings, and release years.

## 🛠 Tools Used
- Power BI
- Microsoft Excel

## 📊 Dashboard Features
- Total shows & directors overview
- Movies vs TV Shows distribution
- Content growth trends over time
- Global distribution and top genres

## 📈 Key Insights
- TV Shows drive recent content growth
- Drama and Comedy dominate the platform
- The US and India lead in content production
- Mature-rated content forms a large portion

## 📂 Files Included
- Netflix_Dashboard.pbix → Power BI file
- Netflix_Dataset.xlsx → Dataset
- /screenshots → Dashboard previews
